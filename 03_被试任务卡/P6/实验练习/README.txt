实验练习

本练习用于熟悉两个系统的实验流程，不计入正式实验结果。

请先完成练习一，再完成练习二。

练习一：RefineUI

第一步：打开 RefineUI。

第二步：传入练习材料。
- 传入 UI 截图，文件名：R组练习UI的截图.png。
- 传入 JSON 文件，文件名：R组练习UI的JSON文件.json。
- RefineUI 练习不需要提示词文件。

第三步：查看系统输出。
- 阅读系统给出的 UI 评估建议。
- 可把练习输出保存为：R组练习系统输出.txt。

第四步：打开 Figma 文件进行尝试修改。
- 打开 Figma 文件，文件名：R组练习UI的Figma文件.fig。
- 尝试根据系统建议修改 UI。
- 修改稿可保存为：R组练习参与者修改稿.fig。

练习二：GPT Baseline

第一步：新建一次全新的 GPT 对话。

第二步：传入练习材料。
- 传入 UI 截图，文件名：G组练习UI的截图.png。
- 传入 JSON 文件，文件名：G组练习UI的JSON文件.json。
- 使用提示词，文件名：G组练习UI的提示词.txt。

第三步：查看系统输出。
- 只生成一次回答，不继续追问。
- 阅读系统给出的 UI 评估建议。
- 可把练习输出保存为：G组练习系统输出.txt。

第四步：打开 Figma 文件进行尝试修改。
- 打开 Figma 文件，文件名：G组练习UI的Figma文件.fig。
- 尝试根据系统建议修改 UI。
- 修改稿可保存为：G组练习参与者修改稿.fig。

两个系统都练习完成后，请返回上一级文件夹，继续进行第一次实验。
